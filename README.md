# Coursera Algorithmic Toolbox

This repo contains my solutions to serveral of homework problems to UC Davis Online Course the [Algorithmic Toolbox](https://www.coursera.org/learn/algorithmic-toolbox/)  offered on Coursera. All of my solutions are in Python 3 and can be found in the **solutions** folder. Homework prompts and source files can be found in the **starterFiles** folder.
